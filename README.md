# CineVerse

CineVerse is a modern movie discovery web application built with Next.js. It allows users to explore trending movies, search for titles, browse categories, and save their favorite movies.

## Live Demo

https://cineverse-next.vercel.app

## Features

- Browse popular, top-rated, and upcoming movies
- Search movies by title
- Browse movies by category
- Add and remove favorites
- Responsive design
- Dynamic data fetching from TMDB API
- Load more movies

## Technologies

- Next.js
- JavaScript
- React
- Tailwind CSS
- TMDB API
- Axios

## Getting Started

```bash
npm install
npm run dev
```
